# Simple Web3 Transaction Tracker

This is a beginner-friendly blockchain project.

This project helps you:
- Connect a crypto wallet
- Read wallet address
- Check blockchain network
- Fetch recent transactions
- Display transaction list

This project uses:
- HTML
- JavaScript
- Public blockchain API

## Features
- One-click wallet connect
- Show wallet address
- Show network name
- Fetch recent transactions
- Simple transaction list

## Requirements
- MetaMask installed
- Internet connection
- Basic JavaScript knowledge

## How to Run
1. Download this repository
2. Open `index.html` in browser
3. Click "Connect Wallet"
4. Click "Load Transactions"
5. See transactions on screen

## File Structure
- index.html -> Main webpage
- app.js -> App logic
- config.js -> API settings
- provider.js -> Web3 provider
- connect.js -> Wallet connect
- wallet.js -> Wallet reader
- network.js -> Network reader
- tx.js -> Transaction fetch logic
- ui.js -> Screen updates
- utils.js -> Helper functions

## Learning Goal
This project helps beginners learn:
- How blockchain transactions work
- How to read wallet history
- How blockchain APIs are used
